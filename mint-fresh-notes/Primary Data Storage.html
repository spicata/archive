Primary Data Storage.md

Last modified: 2023-07-10

Table of Contents

Primary storage is directly accessed by the CPU. There is RAM1, which is volatile, and used to store temporary information. There is ROM2 that contains crucial firmware such as BIOS. It is not volatile.

Hierarchy

Cache is looked to first. Then RAM. Then secondary storage.
