Downloading Prerequisites

  1. sudo apt-get install ruby-full build-essential zlib1g-dev
  2. Get files from https://rubygems.org/pages/download, extract, then (when in that directory), do ruby setup.rb
    • May have to sudo mode
  3. GCC and Make are prebuilt on most Linux distos (phew)

Downloading Jekyll

  1. gem install jekyll bundler
    • May also need to sudo mode
